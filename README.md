# AI4EDA

一个用于学习 AI 辅助数字电路设计的实践仓库。

## 目标

探索如何利用 AI 技术提升数字 IC 设计效率，包括：

- RTL 代码生成与优化
- 设计规范检查与自动修正
- 测试激励自动生成
- 文档自动生成与维护

## 内容

- `rtl/` — 数字电路 RTL 设计（SystemVerilog）
- `tb/` — 验证测试平台
- `sim/` — 仿真脚本
- `docs/` — 架构文档与设计笔记

## 工具链

- 仿真：Verilator / VCS / ModelSim
- 综合：Design Compiler
- 形式验证：JasperGold

## 快速开始

克隆仓库：
```bash
git clone https://github.com/lgwd666/AI4EDA.git
cd AI4EDA
```
